Basic Savings

Believe you can save with just
a barangay certificate.

Start saving with the RCBC Basic Savings

RCBC Basic Deposit Account is a savings account that caters to all walks of life.
Open an account with a barangay certificate or 1 valid ID
and an initial deposit of Php 100.

This is perfect for people who:

  • Have no current bank account

  • Don't have a primary valid ID



  • What You're In For:

    • Initial deposit

      PHP 100

    • Average Daily Balance (ADB) Requirement

      None

    • Maximum Account Balance

      PHP 50,000

    • Interest Rate

      None

    • Fund Access

      • Over The Counter
      • ATM
      • RCBC Online Banking
      • RCBC MyDebit Card

  • You Only Need to Prepare:

    • Completed RCBC account
      opening forms

    • 1 valid ID or a barangay certificate
